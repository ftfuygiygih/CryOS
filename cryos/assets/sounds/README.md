# CryOS — Звуки системы

Папка для системных звуковых файлов (OGG Vorbis).

| Файл       | Когда играет                        |
|------------|-------------------------------------|
| `hi.ogg`   | Сразу после входа в систему         |

## Требования к файлам
- Формат: **OGG Vorbis** (`.ogg`)
- Частота дискретизации: 44 100 Гц или 48 000 Гц
- Каналы: стерео или моно

## Как воспроизводится
`session.py` перебирает доступные плееры в порядке предпочтения:
`paplay` → `pw-play` → `aplay` → `ffplay` → `mpg123`

Воспроизведение неблокирующее — сессия стартует параллельно.
