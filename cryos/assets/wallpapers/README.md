# Обои CryOS

Поместите сюда файлы обоев:

```
wallpapers/
├── w01.png   ← Обои 1 (отображается первой по умолчанию)
├── w02.png   ← Обои 2
└── w03.png   ← Обои 3
```

## Требования
- Формат: PNG (предпочтительно), JPG, BMP
- Разрешение: рекомендуется 1920×1080 или выше
- Имена строго: `w01.png`, `w02.png`, `w03.png`

## Выбор обоев
Обои выбираются в рабочем столе через меню правой кнопки мыши
или через настройки (`cryos-settings`).

Текущий выбор сохраняется в `~/.config/cryos/wallpaper.conf`.
