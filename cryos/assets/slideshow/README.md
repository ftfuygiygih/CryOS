# Слайды приветствия CryOS (OOBE Slideshow)

Поместите сюда 10 изображений слайд-шоу:

```
slideshow/
├── 01.png   ← Слайд 1  (приветствие / логотип)
├── 02.png   ← Слайд 2
├── 03.png   ← Слайд 3
├── 04.png   ← Слайд 4
├── 05.png   ← Слайд 5
├── 06.png   ← Слайд 6
├── 07.png   ← Слайд 7
├── 08.png   ← Слайд 8
├── 09.png   ← Слайд 9
└── 10.png   ← Слайд 10 (финальный / «Начать»)
```

## Требования
- Формат: PNG
- Разрешение: 900×600 или 1280×720
- Имена строго: `01.png` … `10.png`

## Поведение
Слайды показываются последовательно при первом запуске CryOS.
Переход: кнопка «Далее» или автоматически каждые 5 секунд.
Последний слайд содержит кнопку «Начать настройку».
